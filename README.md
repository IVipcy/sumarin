# ようやくチェッカー ✨

小4向けの要約（ようやく）評価アプリ。
光村図書 国語4年「**アップとルーズで伝える**」（中谷日出）の要約を、教科書「要約するとき」の基準でABCD評価します。

## 評価対象

- **原文**：「アップとルーズで伝える」（サーバー側に組み込み済み）
- **入力**：子どもが書いた要約のみ

## 評価軸（教科書より）

| 評価軸 | 内容 |
|--------|------|
| 要点の選択 | 中心となる文・くり返し出る言葉（アップ・ルーズ）を使えているか |
| 知らない人への伝わりやすさ | 読んでいない人にも正しく伝わるか |
| 事実と感想の区別 | 元の文章と自分の感想を分けられているか |
| 構成・文のつながり | 文同士が論理的につながっているか |

## ローカルで動かす

```bash
npm install
cp .env.example .env
# .env に OPENAI_API_KEY を設定
npm start
```

http://localhost:3000 を開く

## Render へのデプロイ

1. GitHub にリポジトリを push
2. [Render](https://render.com) で New → Web Service
3. リポジトリを選択
4. 環境変数 `OPENAI_API_KEY` を設定
5. Deploy

## 使い方

1. 書いた要約を入力（または .txt ファイルをアップロード）
2. 「評価する！」ボタンを押す
3. ABCD評価・評価軸ごとのフィードバック・修正例が表示される
